# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Core library `@403xproof/core` for proof generation and verification
- ProofGenerator class for creating cryptographic proofs
- ProofVerifier class for verifying proofs
- Browser wallet support (Phantom, Backpack, Solflare)
- Solana Keypair support for Node.js
- Timestamp validation
- Wallet address verification
- TypeScript definitions
- Full documentation

### Features
- Cryptographic proof generation from Solana signatures
- Tamper-proof proof structure
- Signature verification
- Timestamp validation
- Cross-platform support (browser and Node.js)
